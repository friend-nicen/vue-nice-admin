
1. 代表页面的组件一般以文件夹的形式通过index.js导出组件，方便观察层次结构，并且页面组件一般都会拆分JS模块，通过文件夹也更加方便文件的分类，保持目录的简洁。
2. 其他的组件，如果设计到大量的逻辑，需要拆分JS模块，可以用文件夹，如何很简单的直接用.vue文件即可。
3. 如何让父子组件的层级更加清晰？首先名字可以按层级写
4. 名字较长的组件用“-”分割，更加友好
